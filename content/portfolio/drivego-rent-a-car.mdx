## Project Context

DriveGo required an end-to-end web portal to digitize vehicle reservations, customer verification, and online security deposits.

## Development & Security Integration

Developed MERN / Next.js stack application featuring serverless API routes, JWT authentication, and interactive vehicle filtering UI. Audited booking API parameters against race conditions and unauthorized pricing override attacks.

## Security Hardening

Server-side price calculation enforcement and atomic database reservation locking.

## Business Results

Reduced booking processing time by 80% and successfully processed over 1,500 reservations with zero payment security incidents.
